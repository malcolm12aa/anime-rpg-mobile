# v1.1.3 — Ability Balance & Basic Ability Scaling

## Goals

This update performs a full ability cleanup pass:

- Remove **Physical** as an ability element.
- Convert old physical abilities into **martial / weapon techniques** assigned across the six real elements.
- Normalize ability scaling and bonus stat labels to the game's Basic Abilities.
- Add enough new abilities to make skills and spells evenly balanced by element and rank.

## Six Ability Elements

Skills and spells now use these six elements:

- Fire
- Ice
- Lightning
- Wind
- Earth
- Water

Old Light, Dark, Arcane, Poison, and Physical entries are normalized for balance:

- Light / Holy / Radiant → Fire lane
- Dark / Shadow / Void → Ice lane
- Arcane / Mana → Lightning lane
- Poison / Nature / Plant → Earth lane
- Physical / Martial / Weapon / Neutral → assigned across the six elements as martial techniques

## Basic Ability Scaling

Ability scaling now uses only:

- Strength
- Endurance
- Dexterity
- Agility
- Magic

Old labels such as STR, INT, WIS, CON, DEX, and CHA are converted before display and scaling.
