# v0.9.6 — Leveling Passive Traits & Mastery Bonuses

This update adds unique leveling passives for every race/race evolution and every job/job upgrade.

## What changed

- Every race now receives a one-of-a-kind Passive Trait generated from its name, category, tier, and build focus.
- Every job now receives a one-of-a-kind Passive Mastery Bonus generated from its name, role, tier, and build focus.
- Passive bonuses scale from both the class stage level and the player's total character level.
- Passive stats feed into the Basic Ability background totals, so the build grows even when visible stage values reset after evolutions/upgrades.
- The Status / Class screen now displays a Leveling Passives section showing active passive names, current bonuses, and scaling text.

## Balance note

Passives use small legacy stat bonuses that are converted into Basic Ability growth. They are designed to matter over time without replacing gear, class levels, title bonuses, or set bonuses.
