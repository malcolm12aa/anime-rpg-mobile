# v1.1.2.1 — Legend Rotation Quest/Achievement Hotfix

This hotfix repairs the background Legend Engine after the Battle UI update.

## Fixed

- Generated quests now refill directly inside the Quest Board when the board opens.
- Generated achievement/title goals now refill directly inside Achievements & Titles when the tab opens.
- Old saves with empty Legend Engine arrays are safely repaired without resetting the character.
- Completed generated quests still grant rewards and rotate into new contracts.
- Completed generated achievements still unlock generated titles and rotate into new goals.
- Race/job weapon and ability prescription objects are now parsed correctly instead of turning into `[object Object]` names.
- Added a rotation lock so screen refreshes cannot cause duplicate generation loops.

## Added

- `Repair / Refill Goals` button on Quest Board.
- `Repair / Refill Goals` button on Achievements & Titles.

These buttons are mainly for old saves or testing. The system still runs automatically in the background.

## Tested

- New save with empty Legend Engine data.
- Quest Board auto-fill.
- Achievements auto-fill.
- Generated quest replacement after completion.
- Generated achievement display.
- No `[Object Object]` generated quest or achievement names.
- Full JavaScript syntax check.
