# v0.9.9 — Legend Engine Report

This update adds a lightweight in-game AI-director style system for Build Your Legend. It does not use a large external model, so it remains GitHub Pages friendly and works fully inside the browser.

## Added

- Dynamic quest generation based on the current player build.
- Dynamic achievement and title generation.
- Generated title bonuses that can be equipped like normal achievement titles.
- Progress tracking for ability uses, elemental uses, spell uses, and equipped weapon battle progress.
- A new Legend Engine screen under the Adventure navigation group.
- Quest Board link to the Legend Engine.

## Main Files

- `js/data/legend-engine.js`
- `js/systems/legend-engine.js`
- `js/ui/screens.js`
- `js/ui/components.js`
- `js/core/game-loop.js`
- `js/core/state.js`
- `js/systems/battle.js`
- `js/systems/leveling.js`

## Static Hosting Note

This is template/procedural AI instead of an external API model. That keeps the game lightweight, mobile-friendly, offline-friendly, and safe to publish on GitHub Pages.
