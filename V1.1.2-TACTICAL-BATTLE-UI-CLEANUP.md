# v1.1.2 — Tactical Battle UI Cleanup

This update rebuilds the Battle UI so combat is easier to read and play without removing any existing combat systems.

## Added

- Two-column desktop battle board.
- Compact enemy card with expandable enemy details.
- Clear enemy intent card with tactical recommendations.
- Compact player resource card.
- Compact ally cards with selectable ally tactics.
- Action tabs: Recommended, Attack, Skills, Spells, Items, Defend, Allies.
- Compact battle ability cards.
- Damage preview and element matchup preview.
- Weapon and Element Mastery requirement warnings.
- Defend action.
- Flee battle action.
- Boss phase panel with threshold markers.
- Weakness-window guidance for bosses.
- Status tooltips and status help panels.
- Collapsible Battle Log.
- Separate Battle Result screen for victory, boss victory, escape, and defeat.
- Mobile sticky battle action panel.

## Ally Tactics

- Auto: allies follow their normal role behavior.
- Guard You: allies prioritize protecting the player and maintaining Guard.
- Burst: allies deal more damage during automatic ally actions.
- Support: allies are more likely to heal and apply Focus.

## Compatibility

Existing saves remain compatible. The update only adds optional UI state fields and battle flow polish.
