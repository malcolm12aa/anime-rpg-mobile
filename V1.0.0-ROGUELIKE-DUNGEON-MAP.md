# v1.0.0 — Roguelike Dungeon Map Update

This update turns the Map / Dungeon tab into a route-board style roguelike loop.

## Added

- Node-based route choices instead of only Explore Next Floor.
- Biomes every 10 floors:
  - Goblin Warrens
  - Ashwood Ruins
  - Frostgrave Halls
  - Stormspire Depths
  - Hollow Cathedral
  - Dragonbone Vault
  - Abyssal Library
  - Celestial Battlefield
  - Void-Touched Labyrinth
  - Throne of World Legends
- Dungeon node types:
  - Battle
  - Elite Battle
  - Boss Gate
  - Treasure Chest
  - Wandering Merchant
  - Rest Camp
  - Ancient Shrine
  - Cursed Altar
  - Training Master
  - Race Trial
  - Class Trial
  - Secret Path
  - Abandoned Forge
  - Alchemy Grove
- Danger Meter with low, medium, high, and extreme states.
- Dungeon supplies:
  - Torchlight
  - Keys
  - Rations
  - Scout Tokens
  - Ward Stones
  - Lockpicks
- Event choices with costs, rewards, danger changes, damage risks, supplies, blessings, curses, and battle options.
- Run goals and run summary tracking.
- Boss floors now extend through floor 100.

## Compatibility

Existing saves are hydrated with the new run fields automatically. IDs for races, jobs, items, abilities, quests, and achievements were not changed.
