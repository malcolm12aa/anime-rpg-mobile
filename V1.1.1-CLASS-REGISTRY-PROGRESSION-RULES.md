# v1.1.1 — Class Registry & Progression Rules

## Added

- Full Class Registry coverage for races, race evolutions, jobs, job upgrades, skills, spells, passives, and abilities.
- Hidden/secret entries now display their actual requirements instead of concealing them.
- Ability registry cards show cost, cooldown, power, element, effects, Basic Ability scaling, weapon requirements, and Element Mastery gates.
- Add New Basic Race and Add New Basic Job panels on the Status / Class screen.

## Progression Rules

- Only the current race stage can be leveled with race class points.
- Only the current job stage can be leveled with job class points.
- A new basic race can only be added when the current race stage is maxed.
- A new basic job can only be added when the current job stage is maxed.
- Race evolutions can only be chosen from the current maxed race stage.
- Job upgrades can only be chosen from the current maxed job stage.
- Existing IDs stay stable for save compatibility.
