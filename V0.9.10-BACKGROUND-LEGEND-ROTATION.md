# v0.9.10 — Background Legend Rotation

## What changed

The Legend Engine is now a background system instead of a manual screen.

## Quest Board behavior

- The game keeps up to 6 generated Legend quests active.
- Generated quests appear inside the normal Quest Board categories.
- When a generated quest is completed, the reward is granted automatically.
- The completed generated quest is archived and replaced by a new generated quest.
- New generated quests use a progress baseline, so replacements require new progress instead of instantly completing from old stats.

## Achievement behavior

- The game keeps up to 6 generated achievement/title goals active.
- Generated goals appear in the Achievements & Titles tab.
- When a generated achievement is completed, it unlocks a title with a stat bonus.
- The completed generated achievement is archived and replaced by a new generated goal.
- Unlocked generated titles stay visible in the Achievements & Titles tab and can be equipped.

## UI behavior

- Removed the manual Legend Engine button from the main navigation.
- Removed manual generation buttons from the Quest Board and Achievements screen.
- Existing saves that still point to the old Legend Engine route are safely redirected to the Quest Board.
